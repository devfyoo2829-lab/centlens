# 화면 설계

CentLens의 사용자 흐름을 4개의 페이지로 구성한다. 각 페이지는 채용공고에 명시된 직무 활동에 직접 매핑된다.

---

## 페이지 구조

| 페이지 | URL | 매핑되는 직무 활동 |
|---|---|---|
| 1. 업로드 및 분석 | `/upload` | 신규 광고 소재 발굴, 경쟁사 광고 소재 벤치마킹, 시장 트렌드 분석 |
| 2. 영상 상세 | `/video/{id}` | 광고 크리에이티브 분석 |
| 3. 누적 데이터 | `/archive` | 광고 소재의 과거 성과 분석 |

---

```
│                                                                │
```

### 핵심 인터랙션

- 파일 업로드 또는 URL 붙여넣기 모두 지원
- 분류 드롭다운은 채용공고 4개 업무 중 3개에 직접 매핑 (신규/경쟁사/트렌드)
- 분석 진행 상황은 LangGraph 노드별로 실시간 시각화 (astream_events 활용)
- 완료 후 영상 상세 페이지로 자동 이동

```
│  마케터 코멘트                                                 │

│  │                                                          │  │
│  └────────────────────────────────────────────────────────┘  │
│  AI 진단과 다른 의견이 있다면 (선택)                           │
│  ┌────────────────────────────────────────────────────────┐  │
│  │                                                          │  │
│  └────────────────────────────────────────────────────────┘  │
│                                                                │
│              [  저장  ]                                        │
│                                                                │
└──────────────────────────────────────────────────────────────┘
```

### 핵심 인터랙션

- 6축 게이지 옆에 평가 근거 텍스트 (한국어 1~2문장, "AI 가설" 뱃지 부여)
- 종합 등급은 색상으로 표시 (우수: 녹색 / 보통: 노란색 / 약함: 회색)
- "노드 실행 트레이스 보기" 펼치면:
  - 각 노드의 실행 시간 (ms)
  - 입출력 토큰 수
  - A/B 점수 비교 표 (Movement A: 4.5 / B: 4.5 / 평균: 4.5)
- 마케터 코멘트는 자유 입력. AI 진단과 다른 의견 사유는 선택 사항이지만 학습 데이터로 누적

### 핵심 인터랙션

- 자연어 검색은 OpenAI Embeddings + pgvector 코사인 유사도로 동작

- 카드 클릭 → 페이지 2 영상 상세로 이동
- 카드 정보: 썸네일 + 게임명 + 분류 뱃지 + 종합 등급 + 총점

## 디자인 토큰

### 색상

| 용도 | 라이트 | 다크 |
|---|---|---|
| 우수 (strong) | #10b981 (Emerald 500) | #34d399 |
| 보통 (medium) | #f59e0b (Amber 500) | #fbbf24 |
| 약함 (weak) | #6b7280 (Gray 500) | #9ca3af |
| 변화 축 그룹 | Purple ramp | — |
| 연출 축 그룹 | Teal ramp | — |
| 교차 검증 | Amber ramp | — |
| 전처리/저장 | Gray ramp | — |
| AI 가설 뱃지 | Blue ramp | — |

### 타이포그래피

- 본문: Pretendard 또는 Apple SD Gothic Neo, 14px / 1.6
- 헤더: 18~22px / 1.4 / weight 500
- 캡션: 12px / 1.4 / muted color

### 간격

- 페이지 패딩: 24px
- 카드 간격: 16px
- 섹션 간격: 32px

## UX 원칙

1. **한국어 우선** — 모든 라벨/버튼/메시지 한국어
2. **AI 가설 시각화** — AI 출력에는 "AI 가설" 뱃지를 부여하여 마케터의 최종 결정과 시각적으로 분리
3. **노드 진행 시각화** — LangGraph 노드 흐름이 사용자에게도 보이도록 실시간 시각화
4. **점수보다 근거** — 게이지 점수보다 평가 근거 텍스트가 더 도드라지게 배치 (마케터가 점수만 보고 판단하지 않도록)
5. **슈퍼센트 톤** — 차분한 블루/네이비 계열 + 깔끔한 카드 UI
