# 02. 해결 방법

## 제품 설명

광고 컨텐츠 마케터·크리에이티브 디렉터의 직무에 명시된 **"광고 소재의 과거 성과 분석", "신규 광고 소재 발굴", "경쟁사 광고 소재 벤치마킹", "시장 트렌드 분석"** 업무를, 슈퍼센트가 자체 정의한 **변화 3축 + 연출 3축 프레임**으로 자동 분해해주는 분석 어시스턴트.

핵심 원칙은 슈퍼센트가 명시한 운영 철학 그대로다.

> data-driven insights with human oversight

— Pizza Ready UA 전략 글, 2025.05

AI는 영상을 6축으로 분해하고 점수와 근거를 산출하지만, 영상의 활용 결정은 마케터가 한다.

---

## 핵심 기능

### 1. 영상 자동 분해 — 변화 3축 + 연출 3축

Annie 글에 정의된 6축을 그대로 사용한다.

| 축 | 평가 기준 (Annie 글 표현 그대로 인용) |
|---|---|
| 움직임 | 한눈에 눈길을 끄는 변화 — 튕기고, 무너지고, 쏟아지는 장면. 짧고 강한 물리적 반응은 그 자체로 후킹 요소가 되고, 초반 유저를 빠르게 몰입시킵니다. |
| 성장 | 변화하는 플레이어 — 플레이어가 변화하면, 유저는 게임의 흐름을 단번에 파악하게 됩니다. |
| 확장 | 넓어지는 공간, 변화하는 시스템 — 공간이나 시스템이 확장되면 유저는 '더 많은 것을 할 수 있다'는 해방감과 기대감을 느낍니다. |
| 카메라 | 시점을 바꿔 공간을 보여주거나, 슬로우/가속으로 흐름을 조절하거나, 이동하는 카메라로 외부 상황과 정보를 자연스럽게 드러냅니다. |
| 컬러 | 메인 오브젝트에는 선명한 색을, 배경엔 차분한 톤을 써 집중을 유도하고, 위험·금지 상황엔 붉은 계열을 사용해 직관적으로 인식하게 합니다. |
| 사운드 | 변화 순간에 효과음을 맞춰 몰입을 강화하고, 반복 조작에는 짧은 사운드로 리듬을 더하고, 장면 전환에는 공간감 있는 음향으로 몰입을 확장. |

### 2. 종합 평가 — Weakest-link 등급

각 축별 5점 척도 + 평가 근거(한국어 1~2문장). weakest-link 방식으로 종합 등급 산출 — 한 축만 높은 영상이 과대평가되는 것을 방지.

| 등급 | 임계값 |
|---|---|
| 우수 (strong) | 모든 축 ≥ 4 AND 총합 ≥ 24 |
| 보통 (medium) | 모든 축 ≥ 3 AND 총합 ≥ 18 |
| 약함 (weak) | 그 외 |

### 3. 분석 대상 분류

채용공고 직무 정의에 맞춰 3개 분석 대상을 모두 지원:

| 분류 | 용도 |
|---|---|
| 신규 (new) | 사내 제작 또는 외부 의뢰분 사전 검토 |
| 경쟁사 (competitor) | 벤치마킹 분석 |
| 트렌드 (trend) | 글로벌 탑 크리에이티브 분석 |

### 4. 마케터 의사결정 (Human-in-the-loop)

각 영상에 대해 마케터가 다음을 입력:

- 분류 확정 (신규/경쟁사/트렌드)
- 본인의 코멘트
- AI 진단과 다른 의견이 있을 경우 사유

### 5. 누적 분석 자산 + 시맨틱 검색

영상별 6축 점수, 마케터 코멘트, 분석 일자가 Supabase에 누적된다. 자연어 검색으로 과거 분석 자산을 빠르게 조회.

> 예: "확장 축이 강한 시뮬레이션 게임 영상", "사운드 약했던 경쟁사 영상"

이는 채용공고에 명시된 "광고 소재의 과거 성과를 분석하고 향후 실행 계획 수립" 업무에 직접 활용 가능한 데이터 자산이 된다.

---

## 사용자 이용 흐름

```
[1] 영상 업로드 + 분류 태그 (신규 / 경쟁사 / 트렌드) + 게임명 + 장르
        ↓
[2] AI 자동 분해
    프레임 추출 → STT → 6축 채점 (A/B 교차) → weakest-link 등급
        ↓
[3] 결과 확인
    6축 게이지 + 추출 프레임 5장 + 축별 근거 텍스트 + 종합 등급
        ↓
[4] 마케터 코멘트 + 분류 확정
        ↓
[5] 누적 데이터 저장
    이후 시맨틱 검색·필터·비교 가능
```
